# Shield

受[微信红包插件](https://github.com/geeeeeeeeek/WeChatLuckyMoney)启发 (感谢作者 @geeeeeeeeek)，写了一个以前一直想实现的功能 -- 屏蔽微信朋友圈，只是为了好玩而已。

开启此插件后，当用户点击微信的 "发现" 栏后，会自动跳到 "我" 栏。

目前此插件仅在 MX4 Pro 上测试过。
